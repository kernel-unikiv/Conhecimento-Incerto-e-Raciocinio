# Lógica Fuzzy

## Visão Geral
A Lógica Fuzzy permite representar conceitos imprecisos por meio de
valores graduais, ao invés de classificações binárias.

## Conhecimento Incerto
É particularmente adequada para situações onde a incerteza é
linguística ou subjetiva.

## Aplicação na Plataforma ApostaIA
A plataforma utiliza lógica fuzzy para traduzir probabilidades numéricas
em termos compreensíveis para o utilizador, como:
- Alta confiança
- Confiança média
- Baixa confiança

## Exemplo Conceitual
Uma probabilidade de 0.72 pode ser interpretada como "alta", enquanto
0.45 representa "média".
